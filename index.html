Dolmen AR Prototype DEMO

iOS(Quick Look), Android(Scene Viewer), WebXR(가능 단말)에서 동작하는 경량 AR 데모입니다.
아래 dolmen.glb / dolmen.usdz 경로를 실제 파일 경로로 교체하세요.
주의사항
• iOS는 Safari 사용 권장(Quick Look). iOS Chrome은 제한될 수 있습니다.
• 파일명이 다를 경우 src / ios-src를 실제 경로로 수정하세요.
• WebXR은 지원 단말/브라우저에서만 동작합니다.
• 성능 문제 시 폴리곤 수를 줄이고 텍스처 해상도를 낮춰 주세요.
